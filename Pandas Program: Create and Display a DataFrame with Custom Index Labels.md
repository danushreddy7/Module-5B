## ðŸ’» Program
```
import pandas as pd
data = {
    'Name': ['Alice', 'Bob', 'Charlie'],
    'Age': [25, 30, 35],
    'City': ['New York', 'Los Angeles', 'Chicago']
}
index_labels = ['a', 'b', 'c']
df = pd.DataFrame(data, index=index_labels)
print("DataFrame with custom index labels:\n")
print(df)
```
